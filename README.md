# home-lab

## Setup
* Install ubuntu server
* Install microk8s
* enable the following addons:
  * dns
  * helm3
  * metallb
  * metrics-server
  * storage
  * ingress
  * openebs
* Install argocd
* set secret for external-secret


## Services
* argocd
* cert-manager
* external-secrets
* kubevirt
* prometheus
* mealie
